How to navigate the admin panel:

In the admin panel, you can view all the offerings, add new offerings, and edit existing offerings.

Using the left-hand side panel or the search bar, you could search for offerings under the various organizations. You could edit the information of the offering by clicking on the edit button next to the organization name. If you scroll to the bottom of the page, you could view expired listings.

Under offerings, you could view various information about organizations like their phone number, website, and address.

Using the Upload button, you could add new offerings by uploading a CSV file. You could also download all the offerings that are currently listed by using the Download button.

Lastly, you could authorize or deauthorize users in accessing this panel by clicking on the 'Authorize Users' button in the side panel.
